运行结束后，可在路径 **Application.Tree.Data.Results Summary.Run-Status.Output** 下读取以下状态标志，快速判断各类计算成败：

表格

复制

|标志|含义|
|-|-|
|CSSTAT|案例研究（V7.3 前旧功能，现已被 Sensitivity 中的 Cases 取代）|
|CVSTAT|收敛块|
|PCESSTAT|物性估算|
|PPSTAT|物性表|
|PROPSTAT|物性计算|
|RSTAT|Calculator 与 Transfer 块|
|SENSSTAT|Sensitivity 块|

**通用规则**

- **0** = 无错误

- **1** = 错误

- **2** = 警告

- **缺失值**（RMISS、NaN 或对于无警告场景返回 2）= 本次运行未使用该类功能

**PCESSTAT 额外说明**

- 单独运行物性估算时：0/1/2 分别表示成功/错误/警告

- 与模拟一起运行时，规则扩展：
– **10** = 估算警告
– **11** = 模拟成功且估算成功
– 其他末位≠0 的两位数 = 估算成功
– **四位数且首位=1** = 严重错误（首位为严重程度，其余位为错误类型细节）

- 若模拟未包含估算，则值为 NaN
