以下变量/结果**无法**通过自动化接口读取：

- 塔器（Column）分析结果

- 收敛监视器（Convergence Monitor）相关变量

- BatchOp 与 BatchSep 模块的结果

- 批处理流程的 strip-chart 图线及全部时间序列结果

流股结果**可以**访问，但在“Stream Summary”里额外添加的属性可能取不到。
正确的流股结果路径位于：

`\Data\Streams\<流股ID>\Stream Results\Table\`

示例——读取流股 3 的属性集变量 VVSTDMFL（cuft/hr）：

vb

复制

```VBScript
Set node = ap.Tree.FindNode("\Data\Streams\3\Stream Results\Table\VVSTDMFL cuft/hr  #1")
```

确保步骤：

1. 在 **Setup | Report Options | Stream | Property Sets** 里新建或引用包含 VVSTDMFL 的属性集。

2. 重新运行模拟。

3. 用变量浏览器确认该节点已存在，再于代码中访问。
