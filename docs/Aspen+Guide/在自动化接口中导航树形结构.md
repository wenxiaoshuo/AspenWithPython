导航窗格中看到的树形结构，与自动化接口完全一致。

Aspen Plus 把模拟中的所有对象暴露为一棵由 **IHNode** 节点组成的树。

- 树的根节点通过 **HappLS.Tree** 属性获得。

- 每个 **IHNode** 可以有 0 个或多个子节点；其 **Dimension** 属性决定子节点的组织方式。

    - 若 **Dimension = 0**，则该节点为叶子节点，不再有子节点。

- 子节点集合通过 **IHNode.Elements** 属性返回，类型为 **IHNodeCol**（节点集合对象）。
