RadFrac 塔的温度分布是**只需一个标识符**的变量示例。
在 pfdtut 模拟结果中，可在导航窗格：

**模块 → B6 → 分布 → TPFQ**

下以表格形式查看温度分布。

同一数据在 Variable Explorer 中的路径为：

**Root → Data → Blocks → B6 → Output → B_TEMP**

在该节点下共有 15 个叶节点，标签依次为 1‒15，对应各塔板温度。
