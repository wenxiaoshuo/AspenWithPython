

要添加模块，调用 **Blocks 集合的 Add 方法**，参数格式为
`"模块名!模块类型"`。

```VBScript
Dim problem As Happ.IHapp
Dim blocknode As Happ.IHNode

Set problem = CreateObject("Apwn.document")
Set blocknode = problem.Tree.Elements("Data").Elements("Blocks")

blocknode.Elements.Add "B1!RADFRAC"
```

要删除模块，调用 **Blocks 集合的 Remove 方法**，参数为模块名即可：

```VBScript
blocknode.Elements.Remove "B1"
```
