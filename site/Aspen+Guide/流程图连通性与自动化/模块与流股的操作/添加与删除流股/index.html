    添加与删除流股

    要添加流股，调用 Streams 集合的 Add 方法，参数格式为 "流股名!流股类型"（流股类型可省略，默认为 MATERIAL）。

    示例：

    Dim problem As HappLS
Dim streamnode As Happ.IHNode

Set problem = CreateObject("Apwn.document")
Set streamnode = problem.Tree.Elements("Data").Elements("Streams")

streamnode.Elements.Add "S1!MATERIAL"   ' 物料流
streamnode.Elements.Add "S2"            ' 默认 MATERIAL
streamnode.Elements.Add "S3!HEAT"       ' 热流

    要删除流股，调用 Streams 集合的 Remove 方法，参数为流股名即可:

    streamnode.Elements.Remove "S1"
